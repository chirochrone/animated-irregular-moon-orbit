This Python script is intended to be executed via command-line.

The trajectory of a chosen irregular moon is sourced from JPL Horizons, queried via Python's astroquery module.
The 3D positions of the chosen irregular moon are visualized via Python's matplotlib module.
The Python code for this was partially generated via Claude AI, but I have made necessary corrections (e.g. check queries, adjust timestep, speed, etc.) and visual improvements (labels, colors, opacities, etc.) to ensure that the visualization is accurate.
